Study on Source Code Summarization

Many documents we need to understand and use software technologies include code examples. In fact, code examples are often what people are looking for. To decide whether to study a code example in detail, it's useful to have a summary of what it's about, just like for any other document. But most techniques for summarizing "normal" documents don't apply to code. This problem pops up everywhere one looks for code on the Internet. To illustrate the point, let's try to find code that shows how to read the properties in a Hibernate configuration file.

Using Stack Overflow to find code examples, the result folds the code snippets into the text when displaying search results, and provides no summarization when viewing the content of the posts:
